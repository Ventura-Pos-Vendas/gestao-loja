📄 Instruções de Deploy
Como hospedar gratuitamente na nuvem (sem servidor físico).
1. Crie uma conta em https://firebase.google.com
2. Crie um novo projeto
3. Ative Authentication → E-mail/Senha
4. Crie banco Firestore (modo produção)
5. Vá em Configurações do Projeto → Seus apps → App Web
6. Copie as credenciais e cole em Configurações acima
7. Para hospedar: use https://vercel.com ou https://netlify.com
   (faça upload do arquivo HTML — zero custo)
